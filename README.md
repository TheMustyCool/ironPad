## üìú Description

Ironpad is a local-first, file-based project management system. Built with the help of AI, it simplifies task management while keeping your data secure and accessible. The Rust backend uses Axum for efficiency, while the Vue 3 frontend offers a smooth user experience. Ironpad uses markdown files as the database, allowing for easy version control with built-in Git integration.
